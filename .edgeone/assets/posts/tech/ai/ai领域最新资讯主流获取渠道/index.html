AI 领域最新资讯主流获取渠道

核心概念 AI 领域主流资讯获取渠道，覆盖学术论文、技术博客、开源社区、行业报告、社交媒体等。

1. 国内综合资讯（适合快速看行业动态、落地产品、政策）

1.1. 媒体平台

  1. 机器之心
    国内最专业AI垂直媒体，每日推送论文解读、大厂新品、融资、行业深度报告，有公众号、官网、APP、视频号，兼顾产业与技术。
  2. 智东西、硬氪
    侧重AI硬件、端侧AI、机器人、自动驾驶、消费级AI产品，产业落地新闻多。
  3. 量子位
    偏互联网大厂动态（字节、阿里、腾讯、百度、华为）、大模型更新、AI创业、职场资讯，短视频+快讯更新快。
  4. 36氪、虎嗅、钛媒体
    综合科技媒体，AI板块覆盖投融资、商业化、政策、企业战略，适合看商业视角。
  5. DeepTech深科技
    硬核前沿，偏向基础模型、芯片、算力、生物医药AI、学术转化，深度长文多。

1.2. 公众号/社群快讯号（碎片化每日刷）

  • 快讯类：AI前线、新智元、DataWhale、夕小瑶科技说、机器之心Pro
  • 技术实操类：HuggingFace中文、LLM之道、AI炼丹笔记
  • 政策产业：工信部装备工业一司、各地人工智能协会

1.3. 短视频/直播（轻量化资讯）

  • B站：李沐、同济子豪兄、Yannic Kilcher（搬运AI论文速览）、机器之心官方号
  • 抖音/视频号：量子位、智东西、各大模型厂商官方账号，新品发布会直播

1.4. 国内社区 & 论坛

  1. 知乎：AI大模型、深度学习话题，行业从业者实时讨论最新模型评测、行业争议
  2. 掘金、InfoQ：工程师视角，大模型落地、微调、RAG、Agent实战资讯
  3. AIStudio、ModelScope社区：阿里、百度官方社区，国产大模型更新、开源项目首发
  4. 脉脉：一线大厂内部动态、人才、项目进度，一手产业小道消息

2. 海外前沿渠道（全球顶会、OpenAI/Anthropic/谷歌/英伟达最新技术，必看）

2.1. 官方厂商博客（第一手权威信息，无二次加工）

  • OpenAI Blog、Google DeepMind Blog、Meta AI Blog、Anthropic Blog、NVIDIA Developer Blog、Microsoft Azure AI Blog
    大模型新版本、Sora、Gemini、Claude、硬件算力、开源框架更新会第一时间发布。

2.2. 海外垂直媒体

  1. The Verge、TechCrunch：全球AI产品、商业新闻、发布会快讯
  2. VentureBeat AI板块：AI创业、融资、企业级解决方案
  3. Hugging Face Blog：开源大模型、Transformer、新SOTA模型、工具链更新
  4. LessWrong：AI安全、AGI、对齐研究深度讨论社区

2.3. 论文&前沿预印本（技术核心源头，所有新技术起点）

  1. arXiv（重中之重）：https://arxiv.org/ 每日AI新论文，分类cs.AI、cs.LG、cs.CL（大模型）
    工具辅助筛选：arxiv-sanity、arxiv daily邮件订阅
  2. ** Papers with Code**：自动汇总arXiv论文+开源代码，实时更新SOTA榜单
  3. 顶会官网：NeurIPS、ICML、ICLR、CVPR、EMNLP，每年征稿、录取、新成果公示

2.4. 海外资讯聚合 & 播客

  • Twitter/X：关注OpenAI、DeepMind、Yann LeCun、Andrej Karpathy、李飞飞等AI大佬，第一时间爆料
  • Reddit：r/MachineLearning、r/LanguageModels 全球工程师实时讨论新论文
  • 播客：The AI Alignment Podcast、Machine Learning Street Talk（通勤听前沿解读）

3. 高效聚合订阅渠道（一站式汇总，不用逐个刷网站）

  1. 邮件订阅
  • Machine Learning Weekly、The Batch（DeepLearning.AI出品，每周精选AI论文+行业新闻）
  • 机器之心每日快讯、Hugging Face Newsletter
  1. RSS阅读器（最适合重度从业者）
    订阅arXiv、各大AI博客、媒体官网RSS，用Inoreader、Feedly统一聚合推送
  2. AI资讯聚合工具
  • 国内：智谱清言资讯、豆包AI资讯流、机器之心每日速报
  • 海外：Luma AI、Context AI 自动筛选个性化AI前沿内容

4. 开源&开发者渠道（模型、框架、工程落地最新动态）

  1. GitHub Trending：每日看热门AI开源项目（Agent、多模态、微调框架）
  2. Discord社群：HuggingFace、Llama社区、各类开源大模型项目交流群，内测新工具首发
  3. 框架官方社区：PyTorch、TensorFlow、JAX官网论坛、Discord

5. 分人群渠道推荐方案

5.1. 入门/产品/运营（侧重行业、产品、商业）

量子位、机器之心公众号、36氪AI板块、抖音/B站科技短视频、知乎AI话题

5.2. 算法/大模型工程师（技术前沿、论文、开源）

arXiv + Papers with Code、HuggingFace、X大佬动态、InfoQ、Machine Learning Weekly邮件

5.3. 科研学生（顶会、基础创新、SOTA）

arXiv每日订阅、顶会官网、Reddit r/MachineLearning、LessWrong

5.4. 创业/产业投资（融资、落地、政策）

36氪、智东西、DeepTech、脉脉、VentureBeat

5.5. 兼顾国内外、追求高效少刷APP

Feedly RSS订阅 + The Batch周报 + 机器之心每日快讯（每天10分钟看完核心资讯）

6. 补充小技巧

  1. 每天固定10分钟刷聚合资讯，每周精读1-2篇深度论文/行业报告；
  2. X/Twitter关注50位左右AI学者、大厂研究员，新技术会比媒体提前2-3天曝光；
  3. 国内社群、Discord经常有未公开内测模型、新工具一手消息，是媒体渠道的补充。
