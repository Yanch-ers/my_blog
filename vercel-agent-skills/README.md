# Vercel Agent Skills

Qoder 插件，包含来自 [vercel-labs/agent-skills](https://github.com/vercel-labs/agent-skills) 的全部 9 个 Agent Skills。

## 包含的 Skills

| Skill | 描述 |
|-------|------|
| **composition-patterns** | React 组件组合模式：复合组件、状态提升、避免布尔属性膨胀，含 React 19 API 变更 |
| **deploy-to-vercel** | 部署应用到 Vercel，支持预览部署和生产部署 |
| **react-best-practices** | React/Next.js 性能优化最佳实践，70 条规则涵盖 8 个类别 |
| **react-native-skills** | React Native/Expo 最佳实践：列表性能、动画、UI 模式、原生模块 |
| **react-view-transitions** | React View Transition API 动画实现指南：页面过渡、共享元素、路由动画 |
| **vercel-cli-with-tokens** | 使用 Token 认证的 Vercel CLI 部署和管理项目 |
| **vercel-optimize** | Vercel 成本和性能优化审计，基于指标驱动的分析 |
| **web-design-guidelines** | Web 界面设计指南合规审查 |
| **writing-guidelines** | 文档写作风格指南合规审查 |
