## 你好，世界

欢迎来到我的个人博客。这里是我分享技术笔记、生活随笔和思考的地方。

### 为什么写博客

写作是一种很好的学习方式。通过输出倒逼输入，能够更深入地理解一个话题。同时，记录下来的文字也可以帮助到其他人。

### 这个博客的技术栈

- **Astro** — 现代化的静态站点生成器
- **Tailwind CSS** — 原子化 CSS 框架
- **Cloudflare Pages** — 全球 CDN 部署

### 代码示例

```javascript
function greet(name) {
  return `Hello, ${name}! Welcome to my blog.`;
}

console.log(greet('World'));
```

> 写作的意义不在于记录结果，而在于记录思考的过程。

希望这里的内容对你有所帮助。
